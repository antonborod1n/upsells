HOW TO CREATE AND PUBLISH A NEW DOCUMENTATION IN BETTERDOCS

Table of Contents
  1. Step 1: Install & Activate BetterDocs
  2. Step 2: Add New Documentation
  3. Step 3: Publish New Documentation
  4. Final Outcome

BetterDocs lets you create and publish new documentation easily by following some simple step by step guidelines. Let’s follow the instructions below to create and publish new documentation in BetterDocs.

Step 1: Install & Activate BetterDocs

BetterDocs lets you create and publish new documentation easily by following some simple step by step guidelines. Let’s followthe instructions below to create and publish new documentation in BetterDocs.

Step 2: Add New Documentation

Open your publish Dashboard and navigate to BetterDocs>>Add New. Create the Documentation just like any other publish Post/Page.

Once you have the Doc ready you need to assign a Category and Tag to it before publishing. From the right sidebar navigate to Document>>Categories. From there you can assign an existing Category or create a new one. From the same place, you can assign Tags.

Step 3: Publish New Documentation

Once you are done writing the Doc with BetterDocs, hit the Blue Button labeled ‘Publish’ at the top-right corner of the screen.

Final Outcome

By simply following all the above steps, you will be able to create and publish your new documentation easily with BetterDocs on your publish website.

That’s how you can simply create and publish new documentation with BetterDocs just within a few steps.
